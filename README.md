# Lamp in  vagrant

## setup:
* php 5.5
* mysql
* phpmyadmin



###Procedure


1. Install [vagrant] (https://www.vagrantup.com/)  and [virtual box] (https://www.virtualbox.org/)

2. Clone / download

3. Do vagrant up while in the directory

4. Place your projects in the projects folder.
